# Conversor de Moedas JS

Projeto simples desenvolvido em JavaScript para conversão de moedas entre Real (BRL), Dólar (USD) e Euro (EUR).

---

## 🚀 Sobre o Projeto

Este projeto realiza conversões de moedas de forma rápida e prática, permitindo calcular valores entre:

- 🇧🇷 Real (BRL)
- 🇺🇸 Dólar (USD)
- 🇪🇺 Euro (EUR)

---

## ⚙️ Funcionalidades

- Conversão de Real para Dólar
- Conversão de Real para Euro
- Conversão de Dólar para Real
- Conversão de Euro para Real
- Interface simples e intuitiva
- Cálculo instantâneo

---

## 🛠 Tecnologias Utilizadas

- CSS

## 💻 Exemplo de Conversão

```javascript
const dolar = 5.40;
const euro = 6.10;

function converterRealParaDolar(valor) {

}
```
